# Packer

Packer is an open-source tool developed by HashiCorp that streamlines the creation of machine images for multiple platforms from a single configuration file. 

### 1. Initialize Packer Configuration

`packer init .`

### 2. Format and Validate Your Template

`packer fmt .`
`packer validate .`

### 3. Build an Image

`packer build image.pkr.hcl`
